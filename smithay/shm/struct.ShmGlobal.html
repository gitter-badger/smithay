fn new<L>(formats: Vec<Format>, logger: L) -> ShmGlobal where L: Into<Option<Logger>>

Create a new SHM global advertizing given supported formats.

This global will always advertize ARGB8888 and XRGB8888 format as they are required by the protocol. Formats given as argument as additionnaly advertized.

An optionnal slog::Logger can be provided and will be used as a drain for logging. If None is provided, it'll log to slog-stdlog.

fn get_token(&self) -> ShmGlobalToken

Retreive a token from the SHM global.

This can only be called once the ShmGlobal has been added to and event loop and has been initialized. If it is not the case, this method will panic.

This is needed to retrieve the contents of the shm pools and buffers.
